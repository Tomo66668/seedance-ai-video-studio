# 私下快速启动说明

这个压缩包已经带 `.env` 配置文件。请只私下发给可信任的朋友，不要公开上传，因为里面包含 API Key。

## 启动步骤

1. 解压压缩包
2. 打开终端进入项目目录
3. 安装依赖

```bash
npm install
```

4. 启动工具

```bash
npm run dev
```

5. 浏览器打开

```text
http://localhost:5173/
```

## 生成文件位置

生成的视频和图片会保存到：

```text
outputs/
```

多项目模式下会保存到：

```text
outputs/项目名/
```

## 注意

不要把 `.env` 发到 GitHub，也不要公开分享这个私密压缩包。
