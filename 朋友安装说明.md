# AI 短剧工作流工具：朋友安装说明

## 1. 安装依赖

先安装 Node.js 20 或以上版本，然后在项目目录执行：

```bash
npm install
```

## 2. 配置 API Key

复制 `.env.example` 为 `.env`：

```bash
cp .env.example .env
```

然后打开 `.env`，填入自己的火山方舟 / Seedance / Seedream API Key。

注意：不要把 `.env` 发给别人，也不要提交到 GitHub。

## 3. 本地启动

```bash
npm run dev
```

启动后浏览器打开：

```text
http://localhost:5173/
```

## 4. 输出位置

生成的视频和图片会保存在：

```text
outputs/
```

多项目模式下，新视频会按项目名保存到：

```text
outputs/项目名/
```

## 5. 线上演示版

GitHub Pages 只能展示前端界面，不能在线真实调用 Seedance API。
